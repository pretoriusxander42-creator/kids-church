## How to Set Up Your Church's Classes

### Step 1: Delete Sample Classes
1. Go to **"Classes"** tab
2. Delete the sample classes you don't need:
   - Click the red "Delete" button on each one
   - Confirm deletion

### Step 2: Add Your Church's Classes
1. Click **"+ Create Class"** button
2. Fill in the details:
   - **Name:** e.g., "Babies & Toddlers"
   - **Description:** e.g., "Ages 0-3 years"
   - **Type:** Select "regular" (or "special" for special needs)
   - **Age Range:** Min and Max ages
   - **Capacity:** Maximum number of children (optional)
   - **Room Location:** e.g., "Room 201"
3. Click **"Create Class"**
4. Repeat for each class

### Example Classes for a Church:
- **Nursery** (0-2 years) - Room 101
- **Preschool** (3-5 years) - Room 102
- **Elementary** (6-10 years) - Room 103
- **Middle School** (11-13 years) - Room 104
- **Special Needs** (All ages) - Room 105
