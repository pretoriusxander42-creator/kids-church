‚ùå TROUBLESHOOTING

Can't Find Child

  • Check spelling
  • Try first name only
  • Add as new child if first visit

Button Doesn't Work

  • Refresh page (Cmd+R)
  • Check internet connection
  • Check terminal for errors

Invalid Security Code

  • Ask parent to double-check
  • Verify all 6 digits
  • Check code wasn't from previous week

Server Not Running

  • Run: ./start-server.sh
  • Wait 30 seconds
  • Refresh browser
