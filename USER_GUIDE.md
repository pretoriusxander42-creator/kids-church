## Checking Out a Child

### Step-by-Step Process:

#### 1. Navigate to Check-out
- Click the **"Check-out"** tab in the dashboard

#### 2. Ask Parent for Security Code
- Request the 6-digit code they received at check-in

#### 3. Enter the Code
- Type or have parent type the 6-digit code
- Code must match exactly

#### 4. Verify and Complete
- Click **"Check Out Child"**
- System will:
  - Verify the code is valid
  - Check that child is currently checked in
  - Display child's name for verification
- **IMPORTANT:** Verify the child's name matches before releasing

#### 5. Confirmation
- Success message appears with child's name
- Give child to parent

### Security Features:
- âœ… Code must match exact check-in
- âœ… Can't check out already checked-out children
- âœ… Invalid codes show clear error message
- âœ… Code is only valid for today's check-ins

### What if Parent Lost the Code?
1. Ask for child's name
2. Go to **Overview** tab
3. Find the child in "Recent Check-ins"
4. Admin can look up the code in the system
5. Verify parent identity before releasing child

## Adding New Children & Parents

### Adding a New Child

#### From Dashboard:
1. Click **"+ Add Child"** button (Overview or Check-in tab)
2. Fill in the form:
   - **First Name** (required)
   - **Last Name** (required)
   - **Date of Birth** (required) - format: YYYY-MM-DD
   - **Gender** (optional)
   - **Allergies** - IMPORTANT for safety
   - **Medical Notes** - Any conditions staff should know
   - **Special Needs** - Check if applicable
   - **Special Needs Details** - Explain requirements
3. Click **"Register Child"**
4. Child is now in the system

### Adding a New Parent

#### From Dashboard:
1. Click **"+ Add Parent"** button
2. Fill in the form:
   - **First Name** (required)
   - **Last Name** (required)
   - **Email** (optional but recommended)
   - **Phone Number** (required) - for emergencies
   - **Address** (optional)
   - **Emergency Contact Name** (optional)
   - **Emergency Contact Phone** (optional)
3. Click **"Register Parent"**

### Linking Parent to Child

#### From Manage Children:
1. Click **"Manage Children"** button
2. Find the child
3. Click **"Link Parent"**
4. Select parent from dropdown
5. Choose relationship type:
   - Mother
   - Father
   - Guardian
   - Other
6. Click **"Link Parent"**

## Troubleshooting

### Common Issues & Solutions

#### "No internet connection" message appears
**Solution:** 
- Check your WiFi/cellular connection
- Try refreshing the page
- Some features may work offline but check-in requires connection

#### Can't log in - "Invalid credentials"
**Solution:**
- Double-check email and password
- Remember password is case-sensitive
- Click "Forgot Password" if needed
- Contact admin if account needs to be created

#### Child not appearing in search
**Solution:**
- Check spelling
- Try searching by last name only
- Child may need to be added first - click "+ Add Child"

#### Security code doesn't work at checkout
**Possible reasons:**
- Code was entered incorrectly - try again carefully
- Child may already be checked out
- Code may be from a previous day
- Contact admin if problem persists

#### Class showing as FULL but there's space
**Solution:**
- Class capacity may need updating
- Contact admin to adjust capacity settings
- Temporarily uncheck a child if needed

#### Dashboard not updating
**Solution:**
- Auto-refresh runs every 30 seconds
- Manually refresh browser (F5 or Cmd+R)
- Check internet connection

#### Print button not working
**Solution:**
- Ensure printer is connected
- Check browser allows printing
- Take photo as alternative
- Parent can screenshot the code

### Error Messages Explained

| Error Message | What it Means | What to Do |
|---------------|---------------|------------|
| "Session expired" | You've been logged out | Log in again |
| "Invalid security code" | Code doesn't match any check-in | Verify code with parent |
| "Child already checked out" | Can't check out twice | Child was already picked up |
| "Class is full" | No more space in selected class | Choose different class or increase capacity |
| "Network error" | No internet connection | Check connection and try again |

## Emergency Procedures

### If a Child Needs to be Retrieved During Service:

1. Staff member notifies you via radio/message
2. Go to **Overview** tab
3. Find child in "Recent Check-ins"
4. Note which class they're in
5. Contact class teacher
6. Have parent verify security code before child is brought out

### If System Goes Down:

**Backup Plan:**
1. Use paper sign-in sheets (keep in binder)
2. Write security codes manually (use random number generator on phone)
3. Keep list organized by class
4. Enter data into system when back online

### Fire Drill / Emergency Evacuation:

1. Teachers take all checked-in children to designated area
2. Parents find children at muster point
3. Verify using emergency paper roster
4. Check off each child as they're reunited with parents
